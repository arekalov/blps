# Спецификация ролей и доступа к API

## Роли

| Роль | Описание |
|------|----------|
| **PUBLIC** | Анонимные пользователи (без аутентификации) |
| **EMPLOYER** | Зарегистрированный работодатель (публикует вакансии) |
| **ADMIN** | Администратор системы |

---

## Матрица доступа к эндпоинтам

### 🔓 Аутентификация и регистрация

| Метод | Эндпоинт | PUBLIC | EMPLOYER | ADMIN | Описание |
|-------|----------|:------:|:--------:|:-----:|----------|
| POST | `/api/v1/auth/register` | ✅ | ✅ | ✅ | Регистрация нового пользователя |

---

### 💼 Вакансии

| Метод | Эндпоинт | PUBLIC | EMPLOYER | ADMIN | Описание |
|-------|----------|:------:|:--------:|:-----:|----------|
| GET | `/api/v1/vacancies` | ✅ | ✅ | ✅ | Получить все вакансии (фильтр по статусу) |
| GET | `/api/v1/vacancies?my=true` | ❌ | ✅ (свои) | ✅ (все) | Получить мои вакансии |
| GET | `/api/v1/vacancies/{id}` | ✅ | ✅ | ✅ | Получить вакансию по ID |
| POST | `/api/v1/vacancies` | ❌ | ✅ | ✅ | Создать вакансию |
| PATCH | `/api/v1/vacancies/{id}` | ❌ | ✅ (свои) | ✅ (все) | Обновить вакансию |
| DELETE | `/api/v1/vacancies/{id}` | ❌ | ✅ (свои) | ✅ (все) | Удалить вакансию |
| PATCH | `/api/v1/vacancies/{id}/tariff` | ❌ | ✅ (свои) | ✅ (все) | Назначить тариф для вакансии |
| PATCH | `/api/v1/vacancies/{id}/publish` | ❌ | ✅ (свои) | ✅ (все) | Опубликовать вакансию (DRAFT → PUBLISHED) |
| PATCH | `/api/v1/vacancies/{id}/archive` | ❌ | ✅ (свои) | ✅ (все) | Архивировать вакансию (PUBLISHED → ARCHIVED) |

---
